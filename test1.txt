What is GitHub used for?
GitHub is a code hosting platform for version control and collaboration. 
It lets you and others work together on projects from anywhere. 
This tutorial teaches you GitHub essentials like repositories,
 branches, commits, and pull requests.

Is GitHub free?
With GitHub Free for user accounts, you can work with
 unlimited collaborators on unlimited public repositories
 with a full feature set, and on unlimited private repositories with
 a limited feature set. With GitHub Free, your user account includes: GitHub Community Support.
